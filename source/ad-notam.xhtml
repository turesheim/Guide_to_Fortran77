Ad Notam

This book was converted from it's original source format which is LaTeX, then to HTML using latex2html. It was then processed with tidy and sed, then converted to EPUB using Eclipse.

I came over this book when I was looking for a good source for doing Fortran 77 development. But since the format was a little awkward and I've become a big fan of EPUB, I decided to use it as an example for the EPUB support I'm implementing for Eclipse's Mylyn Docs.

More information about this process can be found at the public Git repository hosted by GiHub. See https://github.com/turesheim/Guide_to_Fortran77.
